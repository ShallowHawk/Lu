# 📱 移动端使用指南

为了方便您在手机（iOS/Android）上快速更新状态，我们提供了以下两种方案。

## 方案一：使用网页控制台 (推荐)

最简单的方法是使用浏览器访问内置的控制台页面。

1.  **访问地址**: 打开浏览器，访问 `http://您的域名/admin/dashboard`
2.  **验证**: 输入密钥 `my_love_secret_2024` 解锁。
3.  **添加到主屏幕**:
    *   **iOS (Safari)**: 点击底部中间的分享按钮 -> "添加到主屏幕"。
    *   **Android (Chrome)**: 点击右上角菜单 -> "添加到主屏幕"。
4.  **使用**: 点击图标即可像 App 一样打开，点击对应的状态按钮即可一键更新。

---

## 方案二：iOS 快捷指令 (进阶)

如果您使用的是 iPhone，可以通过“快捷指令”App 实现更快的操作（甚至可以用 Siri 语音控制）。

### 1. 创建新指令
打开“快捷指令”App，点击右上角 `+` 号。

### 2. 添加操作 "获取 URL 内容"
搜索并添加操作：`获取 URL 内容` (Get Contents of URL)。

### 3. 配置参数
*   **URL**: `http://101.43.113.154/api/status/update` (请替换为实际 IP/域名)
*   **方法**: `POST`
*   **头部 (Headers)**:
    *   `Content-Type`: `application/json`
*   **请求体 (Request Body)**: 选择 `JSON`
*   **添加字段**:
    *   `secret`: `my_love_secret_2024` (文本)
    *   `user_key`: `mutou` (文本)
    *   `name`: `勤勉` (文本 - 这里可以填您想要的状态)
    *   `description`: `努力工作中...` (文本)

### 4. 保存与运行
*   将快捷指令命名为“我要工作了”或“摸鱼模式”。
*   点击运行测试，若弹出成功提示即配置完成。
*   您可以复制这个指令，修改 `name` 和 `description` 来创建多个不同状态的指令。

## 方案三：Android 自动化 (Tasker / MacroDroid)

如果您想实现“打开王者荣耀自动变成游戏状态”，可以使用 MacroDroid (推荐，操作简单) 或 Tasker。

### 以 MacroDroid 为例：

1.  **新建宏**: 点击“添加宏”。
2.  **触发器 (Triggers)**:
    *   选择 `应用` -> `应用启动/关闭` -> `应用启动`。
    *   选择 `王者荣耀` (或 B站/网易云等)。
3.  **动作 (Actions)**:
    *   选择 `连接` -> `HTTP 请求`。
    *   **URL**: `http://101.43.113.154/api/status/update`
    *   **Method**: `POST`
    *   **Body (JSON)**: `{"secret": "my_love_secret_2024", "user_key": "mutou", "name": "游戏", "description": "王者峡谷激战中"}`
4.  **保存**: 命名为“游戏模式”。

同理，您可以添加“应用关闭”的触发器，将状态改回“在线”或“发呆”。

## 方案四：iOS 自动化 (Automation)

iOS 快捷指令也支持自动化运行，无需手动点击。

1.  打开“快捷指令” App，点击底部的 **“自动化” (Automation)**。
2.  点击右上角 `+` 号，创建 **“个人自动化”**。
3.  **场景举例**:
    *   **App**: 当“哔哩哔哩”打开时 -> 运行快捷指令 -> 更新状态为“看B站”。
    *   **充电器**: 当连接电源时 (晚上) -> 更新状态为“安睡”。
    *   **位置**: 当到达公司 -> 更新状态为“勤勉”。
4.  **注意**: 配置动作时，直接添加“获取 URL 内容”的操作即可（参数参考方案二）。
5.  **去勾选**: “运行前询问”，这样才能全自动静默运行。

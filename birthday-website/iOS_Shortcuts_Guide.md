# iOS 快捷指令配置指南：自动更新状态

这个指南将帮助你创建一个 iOS 快捷指令，让你只需点击一下（或通过 Siri、自动化），就能更新网站上的状态。

## 准备工作

1.  确保你的网站后端已经部署上线，并且外网可以访问（例如：`https://wildmutou.art/api`）。
2.  确保你知道你的更新密钥（代码中默认为 `my_love_secret_2024`）。

## 快捷指令步骤

请在 iPhone 上打开「快捷指令」App，点击右上角 `+` 号创建新指令，然后按以下顺序添加操作：

### 1. 词典 (Dictionary)
*创建一个包含你要发送的数据的词典。*

*   添加 **词典** 操作。
*   添加以下键值对（Key - Text）：
    *   `secret` : `my_love_secret_2024` (你的密钥)
    *   `user_key` : `mutou` (或者是 `qianyu`)
    *   `name` : `忙碌` (你想设置的状态名，比如：休息、工作、运动)
    *   `description` : `正在努力搬砖中...` (状态描述)
    *   `is_online` : `true` (布尔值，表示在线)

> **💡 高级技巧**：你可以把 `name` 和 `description` 设置为「每次运行时询问」，这样你每次运行快捷指令时都可以输入不同的状态。

### 2. 获取 URL 内容 (Get Contents of URL)
*向你的服务器发送数据。*

*   添加 **获取 URL 内容** 操作。
*   **URL**: 输入你的接口地址，例如：`https://wildmutou.art/api/status/update`
*   点击展开 **显示更多 (Show More)**：
    *   **方法 (Method)**: 选择 `POST`
    *   **头部 (Headers)**: 添加新项
        *   Key: `Content-Type`
        *   Value: `application/json`
    *   **请求体 (Request Body)**: 选择 `JSON`
    *   点击 **添加新字段 (Add new field)** -> 选择 **词典 (Dictionary)** -> 选取上一步创建的那个 **词典** 变量。

### 3. 显示通知 (Show Notification) (可选)
*确认状态已更新。*

*   添加 **显示通知** 操作。
*   内容可以填：`状态已更新！`
*   (可选) 你可以将第 2 步的“URL 内容”结果放在通知里，以便调试查看服务器返回的信息。

## 自动化建议 (Automation Ideas)

创建好快捷指令后，你可以在「自动化」标签页中设置自动触发：

1.  **打开特定 App 时**：
    *   例如：当打开「王者荣耀」时 -> 自动更新状态为「游戏」+「峡谷激战中」。
    *   例如：当打开「网易云音乐」时 -> 自动更新状态为「听歌」+「陶醉在音乐里」。

2.  **睡眠模式**：
    *   当「睡眠模式」开启时 -> 自动更新状态为「安睡」+「呼呼大睡中...」。

3.  **充电时**：
    *   当连接充电器时 -> 自动更新状态为「充电」+「回血中...」。

4.  **位置触发**：
    *   当到达「公司」时 -> 自动更新状态为「勤勉」+「努力工作中」。
    *   当到达「家」时 -> 自动更新状态为「休息」+「宅家躺平」。
